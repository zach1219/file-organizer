# 📁 文件整理助手 (File Organizer)

一个基于 Python + tkinter 的桌面文件整理工具，帮助你快速清理杂乱的文件。

## ✨ 功能特性

- **自定义扫描范围** — 添加多个文件夹作为扫描目标
- **两种移动模式**
  - 📄 按文件移动：所有文件按类型（图片/视频/文档/代码等）打散归类
  - 📂 按文件夹移动：保留目录结构，整个子文件夹作为移动单位，散文件按类型归类
- **智能整理建议** — 自动分析文件类型、大小、目录结构，生成分类建议
- **🆕 检测新文件** — 对比上次扫描缓存，自动发现新增文件并给出整理建议
- **单文件撤销** — 可以勾选任意历史移动记录进行精确撤销，不再只能整批回退
- **移动历史** — 完整记录每次移动操作，持久化存储

## 🚀 快速开始

### 环境要求

- Python 3.9+
- 无需安装任何第三方依赖（纯标准库）

### 运行

```bash
python organizer.py
```

### 使用流程

1. 点击「添加文件夹」选择要整理的目录
2. 选择移动模式（按文件 / 按文件夹）
3. 点击「开始扫描」
4. 点击「生成整理建议」查看分类方案
5. 确认后点击「执行移动」
6. 如需撤销，点击「撤销选中记录」精确回退

## 📂 分类规则

| 类别 | 扩展名 |
|------|--------|
| 图片 | jpg, png, gif, bmp, svg, webp, psd, raw, heic... |
| 视频 | mp4, avi, mkv, mov, wmv, flv, webm... |
| 音频 | mp3, wav, flac, aac, ogg, wma, m4a... |
| 文档 | pdf, doc, docx, xls, xlsx, ppt, pptx, txt, md, csv... |
| 压缩包 | zip, rar, 7z, tar, gz, bz2... |
| 代码 | py, js, ts, java, c, cpp, go, rs, html, css, json... |
| 可执行 | exe, msi, dmg, app, deb, rpm... |
| 字体 | ttf, otf, woff, woff2... |
| 数据库 | db, sqlite, sqlite3, mdb... |
| 其他 | 未匹配的文件 |

## 📁 文件说明

| 文件 | 用途 |
|------|------|
| `organizer.py` | 主程序 |
| `move_history.json` | 移动历史记录（自动生成） |
| `scan_cache.json` | 扫描缓存，用于检测新文件（自动生成） |

## ⚠️ 注意事项

- 首次使用建议先在测试文件夹上试用
- 移动前会弹出确认对话框
- 所有移动操作都可撤销
- 重名文件会自动添加 `_1`, `_2` 后缀
- 隐藏文件（`.`开头）会被跳过
